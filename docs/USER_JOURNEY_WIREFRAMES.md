# User Journey and Interface Wireframes

## User Experience Design for ABRSM Grade 8 Piano Scales App

---

## User Personas

### Primary User: Grade 8 Piano Student
- **Age**: 14-18 years old (typical), but can be adult learners
- **Goal**: Prepare for ABRSM Grade 8 examination
- **Technical Proficiency**: Advanced piano skills, moderate computer literacy
- **Practice Environment**: Home practice room with piano/keyboard
- **Needs**: 
  - Quick access to scale requirements
  - Progress tracking to identify weak areas
  - Structured practice routine
  - Audio reference for correct performance

---

## Accessibility Considerations

1. **Keyboard Navigation**
   - Tab through all interactive elements
   - Enter/Space to activate buttons
   - Arrow keys for sliders
   - Escape to close modals

2. **Screen Reader Support**
   - ARIA labels on all controls
   - Meaningful element descriptions
   - Status announcements for dynamic content

3. **Visual Clarity**
   - High contrast mode option
   - Adjustable text size
   - Clear focus indicators
   - Color-blind friendly palette

4. **Motor Accessibility**
   - Large touch targets (minimum 44Ã—44px)
   - No time-limited actions
   - Forgiving click areas
   - Keyboard alternatives for all actions
